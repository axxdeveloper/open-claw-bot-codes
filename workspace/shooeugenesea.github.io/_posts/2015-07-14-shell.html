  1. cd /d %~dp0
    不管執行 script 時候路徑在哪, cd /d %~dp0 後 script 的路徑就到該 script 路徑下
    script, 放在 f:/test/test.bat 
    cd /d %~dp0
dir

    f:\>test\test.bat

f:\>cd /d f:\test\

f:\test>dir
