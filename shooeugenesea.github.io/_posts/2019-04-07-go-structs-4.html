  • Shorthand notation to create and initialize a struct variable
    • Ex. pp := &Point{1,2}
    • It’s same as
      pp := new(Point)
      *pp = Point{1,2}
  • If all fields in struct are comparable, the struct is comparable. (Q: What if struct is not comparable such as slice?)
  • The == compare fields of two struct in order
    • Ex. a := &Point{1,2}
      b := &Point{2,1}
      fmt.Println(a == b) //false
      // same as
      fmt.Println(a.x == b.x && a.y == b.y) //false
