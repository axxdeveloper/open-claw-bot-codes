動機
Mac 螢幕錄影的檔案格式是 .mov（透過 QuickTime 錄製）
可以用 ffmpeg 轉成 gif 來縮小檔案大小

安裝 ffmpeg
brew install ffmpeg

將 mov 轉成 gif
ffmpeg -i AM.mov output.gif
