--- layout: post title: 字串 format date: 2015-09-02 03:41:00 +0800 tags: [python] --- 帶入 dict 的值, 如果用 format api, dictionary 要帶 ** 去展開 key-value 
>>> m = {'a':1.1234, "b":"qq"}
>>> '%(a)1.1f %(a)s %(b)s' %m
'1.1 1.1234 qq'

>>> '{a:1.1f} {a:s} {b:s}'.format(m) # illegal, must use **m

'1.1 qq'
