三種篩選條件：
  • equality filter (ex: “name” = “abc”)
  • multivalue filter (ex. “grade”>5000)
  • sort component of common query pattern (ex. sort({“age”}))

Tricks:
  • 等值篩選的 key 應該放在最前面。
  • 用於排序的 key 應該放在多值欄位之前。
  • 多值篩選的 key 應該放在最後面。（例如：grade > 5000）
