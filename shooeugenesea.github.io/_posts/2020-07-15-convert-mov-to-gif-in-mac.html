--- layout: post title: Convert mov to gif in Mac date: 2020-07-15 03:41:00 +0800 tags: [gif] lang: en ---
Motivation
File extension of screen record in Mac is .mov (By QuickTime)
Can use ffmpeg to convert to gif to reduce file size

Install ffmpeg
brew install ffmpeg

Convert mov to gif
ffmpeg -i AM.mov output.gif
