--- layout: post title: function 參數先排位置再對參數名稱 date: 2015-09-04 15:23:00 +0800 tags: [python] --- 如果沒有指定要丟的參數名稱, python 就會照順序排. 如果順序對不起來就會 error. 
>>> def f(a,b,c):
 print 'a:', a, 'b:', b, 'c:', c

 
>>> f(1,2,3)
a: 1 b: 2 c: 3
>>> f(1,c=3,b=2)
a: 1 b: 2 c: 3
>>> f(1,2,c=3)
a: 1 b: 2 c: 3
>>> f(1,2,b=2) # 第二個參數已經指定了, 所以就不能再指定 b
