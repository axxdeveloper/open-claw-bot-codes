# User Journeys（核心任務路徑）

## Journey 1：建樓與空間配置

1. 進入 Dashboard，點「建立新大樓」。
2. 在建立精靈填入基本資料 + 地下/地上層數。
3. 建立後自動導向「樓層與單位配置」。
4. 在樓層清單查看：樓層、區域、單位數、入住概況。
5. 點「新增單位」進入指定樓層完成配置。

**成功標準：** 不需要理解技術欄位，也知道下一步去哪裡。

---

## Journey 2：住戶與租約串接

1. 到「住戶管理」新增住戶資料。
2. 到「租約管理」選住戶、期間、單位建立租約。
3. 或在樓層頁直接用「快速建立租約」完成同流程。
4. 在租約詳情確認狀態與覆蓋單位。

**成功標準：** 指派住戶到租約建立可在 1~2 個頁面完成。

---

## Journey 3：維運與修繕追蹤

1. 到「業主與持分」建立業主資料。
2. 回樓層頁指派業主持分（時間區間 + 比例）。
3. 到「公共區域管理」建檔常用區域。
4. 在「修繕管理」建立案件並用狀態篩選追進度。

**成功標準：** owners / common area / repairs 都有清楚入口，不再隱藏。

---

## Journey 4：每日巡檢（主管/物業經理）

1. 開 Dashboard 先看待辦與風險摘要。
2. 優先處理：待完成配置、到期租約、進行中修繕。
3. 進入對應頁面直接執行操作，不需要重新找入口。

**成功標準：** 5 秒內理解今天該做什麼。
